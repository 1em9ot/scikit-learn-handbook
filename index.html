AR(1) Process Simulation (例題)

This page demonstrates the AutoRegressive (AR) Model simulation based on the provided conversation example.

Model Definition

The AR(1) process is defined as:

yt = 0.5 * yt-1 + εt

Simulation Results

The following plot shows a single realization of the process (left) and the Autocorrelation Function (ACF) (right). The red dashed line in the ACF plot represents the theoretical decay (0.5k).

Variance Verification

The simulation verifies that the variance of the sample mean in an AR(1) process (with positive correlation) is significantly larger than that of an IID process.

  • Theoretical Ratio (AR/IID): ~3.0 (for T=100, α=0.5)
  • Empirical Result: See console output or notebook.
