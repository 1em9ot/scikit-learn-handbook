検定と精度（分散/標準偏差）のつながり

統計で「精度」を分散（または標準偏差）で測るのは常套手段です。ここでは推定量の精度比較と検定統計量の標準化を一本化して説明します。

1) 「精度」= 推定量の散らばり（MSE）

推定量 θ̂ の良さは基本的に

 MSE(θ̂) = E[(θ̂-θ)^2]
        = Var(θ̂) + Bias(θ̂)^2
  • 不偏なら Bias=0 → MSE = Var
  • 不偏推定量同士なら 分散が小さい方が精度が高い

問14はまさにこれで、ȳ と x̄ は不偏だから精度比較＝分散比較です。

2) 「割る分散/標準偏差」の意味は “標準化”

検定でも推定でも、よく

 (θ̂-θ) / SE(θ̂)   (SE = √Var)

のように標準誤差で割るのは、次の意味づけです。

  • 分子：ズレ（信号）
  • 分母：そのズレがどれくらい普通に出るか（ノイズの尺度）

→ 「何σズレたか」に変換し、比較可能にする。

問14では検定統計量を作っていないだけで、やってることは同じです。Var(x̄) と Var(ȳ) を比べて「どっちがブレないか」を見ています。

3) プーリング推定（2標本比率）で出てくる “D/Var(D)” の接続

記憶にある

  • D（差）を作る
  • D / Var(D) や D / √Var(D) で標準化
  • 分母の Var(D) に p̂（プール）を入れる

これは帰無仮説の下での分散を推定しているだけです。

典型：2標本比率の差

 D = p̂1 - p̂2

帰無 H0: p1 = p2 = p の下では

 Var(D) = p(1-p) (1/n1 + 1/n2)

でも p は未知なので、帰無に沿ったプール推定量

 p̂ = (x1 + x2) / (n1 + n2)

を分母に入れて

 Z = D / √( p̂(1-p̂) (1/n1 + 1/n2) )

ここでやってる核

  • “差” D の大きさを
  • “その差がどれくらい揺れるか” √Var(D) で割って
  • 無次元の尺度（Z）にする

4) 問14への完全対応（同じ型）

問14で比較してるのは推定量がどれだけ揺れるか：

 iid平均: Var(x̄) = σu^2 / 10
 AR(1)平均: Var(ȳ10) = (1/10^2) ΣΣ Cov(yi, yj)

もし問14を“検定っぽく”書けば、

 (ȳT - μ) / √Var(ȳT)

という同じ標準化が出てきます。違いは、プーリング問題では分散が未知だから p̂ 等で推定していただけです。

5) 1行で統一すると

  • 推定：精度 = Var(θ̂)（不偏なら MSE=Var）
  • 検定：(θ̂-θ0)/√Var(θ̂) にして「何σ」を見る
  • プーリング：Var 中の未知母数を帰無仮説に沿って推定して差し込む

必要なら、問14をそのまま「μの検定（Z検定）」に変形して、分母に相関込みの Var(ȳT)が入るのを見せられます（プーリングの分母と完全に同型）。
